<u>Objetivo: </u> Observar a diferença do mesmo dataset com os diferentes formatos de arquivos (CSV, Parquet, JSON). 

<u>Dataset fornecido:</u> pedidos_ecommerce.csv

 - Resultados obtidos quanto ao tamanho: 

pedidos.csv                    569.1 KB

pedidos.json                   1548.8 KB

pedidos.parquet                197.3 KB

 - Resultado dos tempos das três queries :

CSV        73.53 ms

JSON       59.37 ms

Parquet    10.63 ms


#### Ambiente recomendado

 - Python 3.8 ou superior.
 - Esse projeto foi testado com VisualStudio
 - Bibliotecas necessárias: pandas, duckdb, pyarrow
